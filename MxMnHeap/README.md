# MIN/MAX HEAP LIBRARY

## Goal
To implement a heap library capable of doing min and max heap

## Features
* Right now only MIN HEAP properties are applicable
* HeapInit(array, msize)
* HeapInsert(value) 
* HeapExtractMin()
* HeapKeyDecrease(index, value);
* HeapKeyDelete(index)
